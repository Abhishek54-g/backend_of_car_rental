# backend_of_car_rental
this is backend of my rental car project which is I recently created for "rentMycar" website which is a demo, I created an api to fetch boolkings and cars and Dashboard details

This project serves as the backbone of my car rental system, and its development is currently ongoing. I initiated this project around January. It handles core operations such as user login, registration, car booking, selection, and booking status checks. On the administrative side, it enables tasks such as managing car and employee statuses, as well as adding, updating, and deleting car records. I originally developed this project over a period of two weeks in January; however, I am currently in the process of adding numerous additional operations to it.
